# QuickQR

QuickQR is a simple and responsive QR Code Generator built using HTML, CSS, and JavaScript.

Users can enter any text or URL and instantly generate a QR code. The application features a clean user interface, customizable QR sizes, dark/light mode support, and a responsive design that works across devices.

## Technologies Used

* HTML
* CSS
* JavaScript

## Features

* Generate QR codes instantly
* Supports text and URLs
* QR size selection
* Dark/Light mode
* Responsive design
* Recent history storage

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Enter text or a URL and generate a QR code.
